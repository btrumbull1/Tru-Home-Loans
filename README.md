# TRU Home Loans

Production starter configured for GitHub Pages behind Cloudflare.

## Architecture
GoDaddy DNS -> Cloudflare -> GitHub Pages

## Next Steps
1. Push to GitHub.
2. Enable GitHub Pages from `main` / `root`.
3. Add custom domain in GitHub Pages.
4. Point Cloudflare DNS to GitHub Pages.
5. Point GoDaddy nameservers to Cloudflare.

Replace the Apply Online URL with your Arive borrower portal URL.
